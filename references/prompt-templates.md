# Prompt Templates

Use these templates as starting points.

## General summary

```text
Understand this video. Explain what is said and what happens visually over time.

Return:
1. a one-paragraph summary
2. a timeline with timestamps or time ranges
3. key spoken points
4. key visual events
5. visible on-screen text
6. uncertainties
```

## Software demo

```text
Understand this software demo video.

Focus on:
- what the narrator is trying to accomplish
- each important UI transition
- buttons, menus, or panels used
- visible errors, warnings, or confirmations
- any on-screen text that changes the meaning

Return a step-by-step walkthrough with time ranges.
```

## Lecture or tutorial

```text
Understand this lecture/tutorial video.

Focus on:
- the main topic
- the sequence of teaching points
- examples shown on screen
- definitions, formulas, or diagrams that appear
- the relation between speech and visuals

Return concise study notes plus a timeline.
```
